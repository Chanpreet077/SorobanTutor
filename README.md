# SorobanTutor

SorobanTutor is an interactive web application designed to help users learn and practice Soroban (Japanese abacus) arithmetic through guided, animated exercises.

The project focuses on visual learning, step-by-step feedback, and accurate tracking of bead movements to support effective mental math practice.

---

## Features

- Interactive Soroban with animated SVG bead manipulation  
- Guided practice with move-by-move feedback  
- Attempt and bead-state logging for learning analysis  
- Secure user authentication  
- Clean and responsive user interface
